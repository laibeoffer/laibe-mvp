工具展開內容

選取 / 移動畫布 / 縮放
  • 選取物件、移動物件、拉伸 / 縮放、編輯屬性、切換既有 / 新設 / 拆除 / 保留 / 需確認。
  • 手掌模式只拖動畫布，不選取物件；縮放包含放大、縮小、符合畫面、回到 100%。
牆體 / 門窗 / 開口
  • 牆體以線段操作，厚度是 draft 參數；既有牆、承重牆、柱、梁不直接刪除。
  • 門窗必須依附 nodeGraph edge；opening 使用 offset / width，承重牆、柱、梁開口需專業確認。
項目庫 / 材質 / 一鍵標註
  • 項目庫依目前 10 個產品圖層連動，不把木作櫃、系統櫃、廚具、防水當第一層圖層。
  • 油漆桶代表材料分類指定，不做正式材料價格；F8 僅對目前圖層新增 draft 標註 placeholder。
