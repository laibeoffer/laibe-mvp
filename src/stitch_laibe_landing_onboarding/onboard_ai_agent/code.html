Requirement Prompt

需求越清楚，
平面圖與預算越精準。

就像 prompt 一樣，輸入越完整，後面的平面拼圖、預算生成與詢價條件就越不容易失準。你只要把現況、想法、限制與檔案交給 AI 書記官，它會先幫你記成可討論的需求筆記。
