# Contract Attachment Suggestions

These are attachment suggestions only. This Agent does not edit formal legal contract text.

| Suggested Attachment | Purpose | Minimum Contents |
|---|---|---|
| Scope Matrix | Make included work and responsibilities clear | Work item, location, responsible party, included / excluded |
| Drawing Register | Fix the priced drawing basis | Drawing name, SVG / plan version, date, revision note |
| Site Photo Set | Document existing conditions | Photo ID, location, date, note |
| Material Schedule | Make pricing comparable | Item, grade, finish, brand level, substitution rule |
| Work Method Note | Clarify execution responsibility | Demolition, protection, hauling, installation, testing, cleanup |
| Exclusion List | Prevent hidden assumptions | Excluded item, owner responsibility, price impact if added |
| Payment And Acceptance Schedule | Link payment to measurable completion | Milestone, acceptance evidence, payment percentage, retainage |
| Q&A Log | Preserve tender clarifications | Question, answer, date, source, attachment reference |
| Site Constraint Note | Prevent access and schedule disputes | Work hours, access route, elevator, noise, security, neighbor rules |
| Risk Register | Keep known risks visible | Risk, trigger, impact, owner, mitigation, status |
