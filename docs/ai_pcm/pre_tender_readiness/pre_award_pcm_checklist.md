# Pre Award PCM Checklist

This checklist is used after bids are received but before any award recommendation. This Agent may flag readiness issues; it does not approve awards or modify legal contracts.

| Check | Purpose | Required Evidence |
|---|---|---|
| Bid completeness | Confirm every bidder priced the same scope | Bid form, scope matrix, exclusion list |
| Exclusion reconciliation | Identify hidden or conflicting exclusions | Bid exclusions comparison |
| Clarification log | Preserve contractor Q&A and owner-side answers | Q&A log with dates |
| Drawing revision alignment | Confirm bids used the same drawing / SVG version | Drawing register |
| Material equivalence | Compare proposed brands, grades, finishes, and substitutions | Material comparison table |
| Schedule feasibility | Check duration, mobilization, access, and milestone assumptions | Schedule comparison |
| Payment and acceptance alignment | Confirm proposed payment terms match acceptance evidence | Payment milestone table |
| Risk allocation | Identify risks shifted to owner or left unresolved | Risk register |
| Human review flags | Escalate legal, safety, procurement, system, payment, or contract issues | Supervisor review note |
| Attachment completeness | Ensure final tender basis can be attached to contract package | Attachment register |

## Pre Award Status

- `CLEAR_FOR_SUPERVISOR_REVIEW`: No readiness blockers, but supervisor review is still required.
- `CLARIFICATION_REQUIRED`: Bid comparison needs clarification before award discussion.
- `NEEDS_HUMAN_REVIEW`: Legal, safety, procurement, payment, contract, or production-system issue exists.
- `DO_NOT_AWARD_FROM_READINESS_VIEW`: Scope or bid basis is not comparable.
