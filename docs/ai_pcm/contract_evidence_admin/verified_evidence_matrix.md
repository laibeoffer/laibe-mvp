# Verified Evidence Matrix

## Purpose

本 matrix 定義哪些資料在符合 `verified` 條件後可以作正式依據。

目前尚未登錄任何 project-specific verified evidence。

## Verification Criteria

| Evidence Class | Minimum Verification Criteria | Formal Basis After Verified |
|---|---|---:|
| 合約主文 | 版本、簽署或確認紀錄、來源、適用專案均可追溯。 | Yes |
| 合約附件 | 附件版本、合約引用、來源與有效狀態可追溯。 | Yes |
| 預算單 | 版本、合約關聯、金額欄位、雙方確認狀態可追溯。 | Yes |
| 報價單 | 報價來源、版本、接受或納入合約狀態可追溯。 | Conditional |
| 工項明細 | 對應合約、附件、預算或變更單可追溯。 | Yes |
| 材料規格 | 規格版本、材料名稱、型號、品牌或等級可追溯。 | Yes |
| 工法範圍 | 對應工項與合約範圍可追溯。 | Yes |
| 不包含項目 | 對應合約或附件，且未被後續變更取代。 | Yes |
| 假設條件 | 被合約、附件或雙方確認接受，且未被實際條件推翻。 | Conditional |
| 平面圖 / SVG / 量體 | 來源、版本、量測方法與適用範圍可追溯。 | Conditional |
| 現況照片 | 時間、地點、拍攝來源、關聯位置可追溯。 | Support |
| 變更單 | 變更範圍、版本、雙方確認、日期可追溯。 | Yes |
| 驗收紀錄 | 驗收項目、結果、日期、確認人可追溯。 | Yes |
| 付款節點 | 合約付款條件、驗收或里程碑關聯可追溯。 | Yes, evidence only |
| LINE 訊息 | 參與者、時間、上下文、截圖或匯出來源可追溯，且需有合約或雙方確認關聯。 | Support only |
| 平台訊息 | 帳號、時間、內容、平台來源可追溯，且需有合約或雙方確認關聯。 | Conditional |
| 雙方確認紀錄 | 雙方身份、內容、時間、版本、確認方式可追溯。 | Yes |

## Current Verified Matrix
