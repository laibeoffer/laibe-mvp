## Review Method
- 以 `Z:\08-Jacky\plan study` 的 7 張 JPG 為唯一正例來源；沒有在這 7 張圖中看到相似平面符號者，不因名稱、常識或 library 分類而放行。
- 以 `Z:\08-Jacky\svg_blocks\_manifest.csv`、`svg_block_index.csv`、`SVG_BLOCK_LIBRARY_REPORT.md` 只做枚舉與定位，不作為 allow 依據。manifest 標為 usable 的 block 仍需通過視覺比對。
- 在 dedicated worktree 內建立只讀來源的 SVG contact sheet / thumbnail gallery：`docs\plan_puzzle_repair\furniture_block_audit_work\svg_focused_page_*.png`、`svg_unknown_page_*.png`、`svg_all_page_*.png`，再以人工視覺檢查對照 7 張 plan JPG。
- 比對特徵包含 top-view 外輪廓、家具/設備構成、符號比例、是否為平面符號。立面圖、人物、車輛、標註、材質、門窗 core、純裝飾即使 library 中存在，仍不得列入家具包候選。
- `PARTIAL_REVIEW_REQUIRED` 代表相似但有方向、比例、裁切、整組物件、黑填、或 package scope 疑慮；partial 一律不得進入 allowlist。
