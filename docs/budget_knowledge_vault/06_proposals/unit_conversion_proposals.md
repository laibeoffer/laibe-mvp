# Unit Conversion Proposals

These proposals are support-only and do not modify `UnitConversion`.

## Proposal UNIT-PROP-001

Track unit conversion readiness as a handoff question between Raw Candidate, MethodSpec, and Budget Engine Entry rather than resolving it inside the vault.

Required evidence before implementation:

- owning workstream,
- source evidence,
- expected input/output units,
- review status,
- integration entry path.
