# MethodSpec Proposals

These are support proposals only. They are not implementation instructions and do not modify MethodSpec runtime.

## Proposal MS-PROP-001

Keep Requirement Form Window and Plan Puzzle SVG / Quantity Facts Window as explicit context windows in MethodSpec readiness discussions.

Rationale: non-verified context must remain review-required and must not produce formal prices.

Boundary: no `MethodSpecCatalog`, `MaterialSpec`, `LaborRule`, `PricingRule`, or Budget Engine changes.
