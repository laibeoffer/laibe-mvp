# Quote Factory Export Package Notes

## Current Package Role

The export package is support evidence for candidate price-range governance. It is not formal pricing authority.

## Required Evidence

- PR #3 state and commit evidence.
- Export manifest evidence.
- Validator command and result evidence.
- Explicit final acceptance or rejection decision.

## Forbidden Promotion

Do not promote observed prices, candidate prices, `PriceRange`, or display unit prices into formal prices or production `PricingRule`.
