# Owner Guide Question Flow

## 1. 基本案件資訊

- 房屋類型
- 坪數 / 面積
- 屋況
- 所在區域
- 使用目的

## 2. 空間需求

- 客廳
- 臥室
- 廚房
- 浴室
- 收納
- 其他空間

## 3. 風格偏好

- 想要風格
- 不喜歡風格
- 參考圖片狀態

## 4. 預算偏好

- 未知
- 區間
- 上限
- 需 AI 協助估算

This is a preference signal only. It must not become formal pricing.

## 5. 時程偏好

- 探索中
- 一個月內
- 三個月內
- 半年內
- 未知

## 6. 檔案狀態

- 是否已有平面圖
- 是否已有報價單
- 是否已有照片
- 是否需要平面拼圖

## 7. 下一步推薦

- 繼續需求整理
- 進入平面拼圖
- 進入預算預覽
- 進入招標前準備
- 需要人工協助

## Question Loop

Each round should record: stage, question, owner answer, extracted fields, confidence, gap update, next recommended step, and source reference.
